const char** yasm_objfmt_module::dbgfmt_keywords

NULL-terminated list of debug format (yasm_dbgfmt) keywords that are valid to use with this object format. The null debug format (null_dbgfmt, "null") should always be in this list so it's possible to have no debug output.
