Detailed Description

Base structure for an effective address. As with all base structures, must be present as the first element in any yasm_arch implementation of an effective address.

unsigned int yasm_effaddr::nosplit

1 if reg*2 should not be split into reg+reg. (0 if not). This flag indicates (for architectures that support complex effective addresses such as x86) if various types of complex effective addresses can be split into different forms in order to minimize instruction length.

unsigned int yasm_effaddr::strong

1 if effective address is /definitely/ an effective address. This is used in e.g. the GAS parser to differentiate between "expr" (which might or might not be an effective address) and "expr(,1)" (which is definitely an effective address).
