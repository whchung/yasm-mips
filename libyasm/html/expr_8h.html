YASM_LIB_DECL void yasm_expr__order_terms ( yasm_expr *  e)

Reorder terms of e into canonical order. Only reorders if reordering doesn't change meaning of expression. (eg, doesn't reorder SUB). Canonical order: REG, INT, FLOAT, SYM, EXPR. Multiple terms of a single type are kept in the same order as in the original expression.

Parameters
eexpression
Note
Only performs reordering on one level (no recursion).
