size_t yasm_dir_help::off

Offset added to data pointer passed into yasm_dir_helper() before data pointer is given to yasm_dir_help.helper(). This is so that a structure can be passed into yasm_dir_helper() and this can be an offsetof() to point the helper function to a specific structure member.
