If the key is not present in the \hyperlink{struct_h_a_m_t}{H\-A\-M\-T}, inserts it, sets $\ast$replace to 1, and returns the data passed in. If the key is already present and $\ast$replace is 0, deletes the data passed in using deletefunc() and returns the data currently associated with the key. If the key is already present and $\ast$replace is 1, deletes the data currently associated with the key using deletefunc() and replaces it with the data passed in.
